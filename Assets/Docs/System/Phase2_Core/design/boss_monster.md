# 보스 몬스터 설계

> 상위 문서: [Phase 2 설계](../design.md)

Yellow 유닛 Monk와 Pawn을 기반으로 한 2종의 보스 몬스터를 정의한다.
각 보스는 일반 몬스터와 달리 고유한 패턴 시퀀스를 가지며,
게임 진행 일정 시간 이후 맵에 1마리씩 스폰된다.

---

## 등장 조건

| 항목 | 값 |
|------|----|
| 스폰 시각 | 게임 시작 **180초** 이후 |
| 스폰 위치 | 플레이어로부터 **14~18 타일** 거리, 장애물 없는 빈 셀 |
| 동시 존재 수 | 보스 1마리만 활성 (사망 후 240초 뒤 재스폰) |
| 테이밍 가능 여부 | **불가** (처치 시 일반 드롭 없음) |

---

## Boss 1 — YellowMonk (황금 주술사)

### 컨셉

힐러 수도승이 어둠에 물들어 저주 에너지를 쏟아내는 보스.
주로 **제자리를 유지하며 장판·마법진 공격**으로 압박한다.
예고 시간이 길어 회피 가능하지만, 세 패턴이 조합되면 안전 지대가 좁아진다.

### 기본 스탯

| 항목 | 값 |
|------|----|
| 최대 HP | 600 |
| 이동 속도 | 1.5 (일반 몬스터의 60%) |
| 기본 공격 | 없음 (패턴 전용) |
| 감지 범위 | 10 타일 |

### 사용 스프라이트

- **Idle**: `Yellow Units/Monk/Idle.png`
- **Run**: `Yellow Units/Monk/Run.png`
- **패턴 시전 모션**: `Yellow Units/Monk/Heal.png` (재활용)
- **시전 이펙트**: `Yellow Units/Monk/Heal_Effect.png`

---

### 패턴 목록

#### P1 — 추적 장판 (Tracking Zone)

> "플레이어를 향해 서서히 조여오는 저주 에너지."

| 단계 | 내용 | 시간 |
|------|------|------|
| **예고** | 플레이어 위치 중심으로 반투명 빨간 원형 범위 표시. 플레이어가 이동하면 마커도 함께 추적. | 2.5 초 |
| **고정** | 범위가 빨간색으로 확정(이후 추적 중단). | 0.5 초 |
| **폭발** | 확정된 위치에 장판 활성화 → 범위 내 유닛에 데미지. | 1 초 |

- **범위**: 반지름 **2.5 타일** 원형
- **데미지**: 보스 기준 단일 타격 (피격 판정 1회)
- **재사용 대기**: 8초

---

#### P3 — 십자 장판 (Holy Cross)

> "수도승의 금지된 신성 문양이 십자로 뻗어나온다."

| 단계 | 내용 | 시간 |
|------|------|------|
| **예고** | 보스 중심으로 상하좌우 십자 범위 테두리 표시 (흰 점선). | 0.8 초 |
| **활성** | 십자 범위 전체가 순간 활성화 → 데미지. | 0.4 초 |
| **잔상** | 범위가 서서히 사라짐. | 0.3 초 |

- **범위**: 보스 중심 상하좌우 각 **5 타일**, 폭 **1 타일**
- **데미지**: 십자 내 모든 유닛 (멀티 히트 아님, 1회)
- **재사용 대기**: 10초
- **조합 예고**: P4와 연속 사용 시 8방향이 됨

---

#### P5 — 예약 저격 (Curse Mark)

> "플레이어가 서 있던 바로 그 자리에 저주를 남긴다."

| 단계 | 내용 | 시간 |
|------|------|------|
| **마킹** | 플레이어 현재 위치에 빨간 점 예고 마커 표시. | 1.5 초 |
| **폭발** | 해당 위치 **1 타일** 범위에 집중 장판 폭발. | 0.5 초 |

- **범위**: 중심 반지름 **0.8 타일** (한 칸 크기)
- **데미지**: 범위는 작지만 **고데미지** (P1의 1.5배)
- **재사용 대기**: 6초
- **특징**: 작은 범위라 회피 가능하나, P1·P3과 조합 시 안전 지대가 좁아짐

---

### 패턴 시퀀스

```
스폰 → 플레이어 접근 이동
      → [감지 성공] → 정지 + 패턴 루프 시작

패턴 루프 (순서 고정 아님, 랜덤 선택):
  P1(8s 쿨) ↔ P3(10s 쿨) ↔ P5(6s 쿨)
  패턴 사이 1.5초 Idle 휴지
  HP 50% 이하: 각 패턴 쿨다운 -20%
```

## Boss 2 — YellowPawn (황금 광전사)

### 컨셉

마을을 지키던 일꾼이 분노로 폭주한 근접 돌격형 보스.
**빠른 대시와 투사체**로 공격하며, 일반 몬스터보다 훨씬 공격적으로 플레이어를 추적한다.
예고 시간이 짧고 돌진이 빠르기 때문에 지속적인 방향 전환이 중요하다.

### 기본 스탯

| 항목 | 값 |
|------|----|
| 최대 HP | 800 |
| 이동 속도 | 3.0 (일반 몬스터와 동일) |
| 기본 공격 | 근접 단타 (일반 몬스터처럼) |
| 감지 범위 | 12 타일 |

### 사용 스프라이트

- **Idle**: `Yellow Units/Pawn/Pawn_Idle Axe.png` (도끼 버전)
- **Run**: `Yellow Units/Pawn/Pawn_Run Axe.png`
- **공격 모션**: `Yellow Units/Pawn/Pawn_Interact Axe.png` (재활용)

---

### 패턴 목록

#### P2 — 돌진 공격 (Rampage Charge)

> "분노의 기운을 모아 직선으로 폭주한다."

| 단계 | 내용 | 시간 |
|------|------|------|
| **예고** | 보스에서 플레이어 방향으로 화살표 범위 표시 (직사각형). | 1.5 초 |
| **돌진** | 예고 방향으로 고속 돌진. 충돌 시 데미지 + 넉백. 장애물 또는 최대 거리 도달 시 정지. | 0.8 초 |
| **경직** | 돌진 후 짧은 멈춤. | 0.7 초 |

- **거리**: 약 **7 타일** (아처 기본 사거리 수준)
- **폭**: **1.5 타일** (돌진 경로 너비)
- **데미지**: 경로 내 유닛 전부 (멀티 히트)
- **재사용 대기**: 7초
- **특징**: 돌진 방향은 예고 시작 시점의 플레이어 방향으로 고정 (이후 추적 안 함)

---

#### P4 — X자 장판 (Axe Slash)

> "도끼를 두 번 크게 휘둘러 대각선 4방향을 날린다."

| 단계 | 내용 | 시간 |
|------|------|------|
| **예고** | 보스 중심 X자(45°·135°·225°·315°) 범위 테두리 표시. | 0.6 초 |
| **1번 슬래시** | 좌상↔우하 대각선 활성화. | 0.3 초 |
| **0.1초 간격** | | 0.1 초 |
| **2번 슬래시** | 우상↔좌하 대각선 활성화. | 0.3 초 |

- **범위**: 보스 중심 각 대각선 방향 **5 타일**, 폭 **1 타일**
- **데미지**: 두 슬래시 각각 독립 판정 (2회 피격 가능)
- **재사용 대기**: 9초
- **조합 예고**: Monk P3(십자)와 조합 시 8방향 커버

---

#### P6 — 느린 도끼 투척 (Triple Axe Throw)

> "도끼 3개를 연속으로 투척. 느리지만 피하기 어렵다."

| 단계 | 내용 | 시간 |
|------|------|------|
| **1번 투척** | 플레이어 방향으로 투사체 1발. 느린 속도(3 타일/초). | — |
| **0.5초 후 2번 투척** | 1번에서 좌측 15° 방향으로 1발. | — |
| **0.5초 후 3번 투척** | 1번에서 우측 15° 방향으로 1발. | — |
| **투사체 소멸** | 최대 **8 타일** 비행 후 소멸. | — |

- **투사체 크기**: 반지름 **0.4 타일**
- **데미지**: 투사체 1개당 단타. 최대 3회 피격 가능.
- **재사용 대기**: 5초
- **특징**: 3발이 약간씩 퍼져 오기 때문에 좁은 통로에서는 피하기 어려움

---

### 패턴 시퀀스

```
스폰 → 플레이어 적극 추적 (기본 이동 공격 포함)
      → [공격 범위 진입] → 기본 공격 + 패턴 루프

패턴 루프 (랜덤 선택):
  P2(7s 쿨) ↔ P4(9s 쿨) ↔ P6(5s 쿨)
  기본 공격은 별도 쿨다운으로 지속
  HP 50% 이하: 이동 속도 +30%, 패턴 쿨다운 -20%
```

## 공통 비주얼 연출

| 요소 | 내용 |
|------|------|
| **스폰 연출** | 맵 외곽에서 등장 이펙트 + HP 바 등장 |
| **장판 예고색** | 노란색 테두리 → 빨간 채움으로 전환 (활성 직전) |
| **돌진 예고** | 빨간 직사각형 화살표 |
| **투사체** | 회전하는 도끼 스프라이트 (Pawn_Idle Axe.png 재활용 or 별도 이펙트) |
| **피격 이펙트** | 일반 몬스터와 동일 (DamageFlash + HitStop) |
| **사망 연출** | CharacterView.PlayDeathSequence() 동일 사용 |

## 확정 사항

| 항목 | 결정 |
|------|------|
| **보스 스폰 방식** | 두 보스 중 **랜덤 1마리**만 등장. 동시 등장 없음. |
| **HP 바** | PlayPage 상단에 보스 전용 HP 바 UI 배치. 평소 비활성화, 보스 등장 시 활성화. |
| **능력치 관리** | `BossMonsterData` **ScriptableObject**로 모든 수치 외부화. |
| **스폰 경고** | 보스 스폰 직전 화면에 경고 UI(이름 + 아이콘) 표시. |
| **보상 드롭** | 없음 (테이밍 불가, 별도 드롭 없음). |
