# 로그라이크 진행 시스템 — 컨셉 문서

> 작성일: 2026-03-03
> 참고 레퍼런스: Vampire Survivors

---

## 1. 핵심 컨셉

Wild Tamer를 **한 판 30~40분의 로그라이크 서바이벌**로 발전시킨다.

- **한 판 구조:** 제한 없는 시간 내 보스 3마리 순차 처치 → 클리어
- **판간 연속성:** 재화(골드)를 통해 영구 업그레이드를 구매, 다음 플레이에 누적
- **임시 성장:** 플레이 중 레벨업 시 3개 선택지 중 하나를 선택해 당 판에만 적용
- **동기 부여:** 질 때도 경험치·재화 보상이 지급되어 다음 판을 더 강하게 시작

## 3. 재화(골드) 시스템

### 획득
| 조건 | 재화 |
|------|------|
| 도달 레벨 1당 | +10 골드 |
| 생존 스쿼드 멤버 1명당 | +15 골드 |
| 보스 처치 1회당 | +100 골드 |
| 클리어 보너스 | +200 골드 |

> 수치는 밸런스 조정 예정. 한 판 평균 기대 골드 300~600 수준을 목표로.

### 저장
- `UserData.gold` (int) — 판간 영구 보존
- 게임오버/클리어 모두 지급 (패배해도 성장 가능)

---

## 4. 레벨 & 경험치

- **경험치 획득:** 몬스터 처치 시 경험치 드롭 (일반 < 특수 < 보스)
- **레벨업 시:** 임시 업그레이드 선택 UI 표시 (3선택지)
- **레벨은 메모리에만 저장** — 판이 끝나면 초기화

### 경험치 곡선 (예시)
| 레벨 | 필요 경험치 |
|------|------------|
| 1→2 | 50 |
| 2→3 | 110 |
| N→N+1 | 50 × 1.15^(N-1) (지수 증가) |

---

## 5. 업그레이드 구분

| 구분 | 선택 방식 | 저장 위치 | 초기화 |
|------|----------|----------|--------|
| **영구 업그레이드** | 상점에서 골드로 구매 | `UserData` (PlayerPrefs) | 유저가 리셋 시에만 |
| **임시 업그레이드** | 레벨업 시 3선택지 | 메모리 (`RunUpgradeState`) | 판 종료 시 |

---

## 6. 보스 클리어 구조

기존 단판 클리어에서 **3단계 클리어**로 변경.

| 단계 | 등장 시간 | 비고 |
|------|----------|------|
| 보스 1차 | 3분 경과 후 | 현재와 동일한 풀 |
| 보스 2차 | 5분 경과 후 | 보스 풀 or 강화된 보스 |
| 보스 3차 | 7분 경과 후 | 최강 보스, 처치 시 클리어 |

- 보스 전투 중 일반 몬스터 스폰 정지 (기존 동작 유지)
- 보스 처치 후 리스폰 대기 없이 다음 단계 카운트 이어서 진행
- `BossSpawnSystem`에 단계(phase) 개념 추가 필요

## 7. 적 스케일링 (시간 기반)

시간이 지날수록 전장이 점점 압박됨.

| 시간 | 스폰 간격 | 한 번에 스폰 수 | 강화 몬스터 확률 |
|------|----------|----------------|----------------|
| 0분 | 100% (기준) | 1 | 0% |
| 2분 | 85% | 1~2 | 5% |
| 5분 | 70% | 2 | 15% |
| 8분 | 55% | 2~3 | 30% |
| 10분+ | 40% | 3 | 50% |

> "강화 몬스터"는 HP/공격력이 1.5배인 변형 스폰. 별도 스프라이트 색조 변경으로 구분.

## 9. 임시 업그레이드 선택 UI

- 레벨업 시 게임 **일시 정지** 후 팝업 표시
- 3개의 카드 제시 (무작위 가중 추출)
- 이미 5레벨인 업그레이드는 풀에서 제외
- 선택 후 즉시 적용, 게임 재개

### 가중치 예시
- 플레이어 능력: 가중치 3
- 스쿼드: 가중치 2
- 테이밍: 가중치 2
- 환경: 가중치 1 (희귀)

## 10. 구현 분리 가이드

아래 모듈은 서로 독립적으로 구현 가능. 우선순위 순 정렬.

| 우선순위 | 모듈 | 핵심 내용 | 의존성 |
|---------|------|----------|--------|
| 1 | **재화 시스템** | `UserData.gold`, 결산 화면 UI | 없음 |
| 2 | **경험치 / 레벨** | EXP 드롭, 레벨업 이벤트 | 없음 |
| 3 | **임시 업그레이드** | 선택 팝업, RunUpgradeState, 능력치 반영 | 레벨 시스템 |
| 4 | **보스 3단계 클리어** | BossSpawnSystem phase 확장 | 없음 |
| 5 | **적 시간 스케일링** | 스폰 간격/수/강화확률 시간 함수 | 없음 |
| 6 | **영구 업그레이드 상점** | 상점 UI, UserData 업그레이드 저장 | 재화 시스템 |
| 7 | **능력치 반영 시스템** | 업그레이드 수치 → 실제 유닛 스탯 적용 | 임시+영구 업그레이드 |
