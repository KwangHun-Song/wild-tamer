# Wild Tamer Like - 게임 컨셉 문서

## 레퍼런스

- **대상**: 와일드 테이머 (Wild Tamer)
- **핵심 요소**: 쿼터뷰 시점, 동물 군집 제어, 자원 수집 및 테이밍 기반 RPG
- **플랫폼**: PC 또는 모바일 (입력 인터페이스 자유)

## 코어 루프

```
탐험 → 전투 → 테이밍 → 부대 확장 → 탐험 ...
```

## 필수 구현 기능

### 1. 군집 및 부대 시스템 (Core)

- 플레이어를 따라다니는 부대의 부드러운 이동 및 대형 유지

### 2. 전투 및 테이밍

- 플레이어 부대와 야생 몬스터의 자동 교전 시스템
- 몬스터 처치 시 일정 확률로 테이밍 가능

### 3. 몬스터

- **일반 몬스터 (3종)**: 종별로 차별화된 이동 패턴과 공격 사거리 보유
- **보스 몬스터 (2종)**: 특수 스킬(장판, 돌진 등) 패턴 보유

### 4. 맵 시스템 (전장의 안개)

- 미탐색 지역은 어둡게 처리, 플레이어 이동에 따라 시야가 밝혀지는 Fog of War
- 실시간 탐색 현황이 반영되는 미니맵 연동

## 선택적 구현 기능 (가산점)

- **군집 알고리즘 직접 구현**: NavMesh를 사용하지 않고 코드로 직접 구현
- **동물 도감**: 발견하거나 테이밍한 동물의 정보를 확인할 수 있는 UI
- **데이터 영속성**: 플레이어 진행 상황(보유 부대, 탐사 지역 등)을 로컬 저장 및 로드

## 평가 기준

### 기술적 역량

- 다수 개체 연산 최적화 처리
- 객체지향 설계 및 코드 모듈화

### 시스템 완성도

- 플레이어 부대와 적 군집 간의 상호작용
- 타격 판정의 정확성

### 플레이 경험 (UX)

- **전투의 손맛**: 공격/피격 시 역경직, 화면 흔들림, 사운드 타이밍 등의 조화
- **타격감 및 연출**: 테이밍 성공 시 시각적 피드백 등 생동감을 높이는 효과
- **조작감**: 이동 및 군집의 움직임이 유저 의도대로 부드럽게 반응

## 제출물

- **플레이 영상**: 주요 기능 및 전투 연출 확인 영상
- **기술 명세서**: 군집 알고리즘의 원리 및 최적화 방식 요약
- **Unity 프로젝트 원본**: 소스코드 포함 (Library, Temp 등 제외)
- **빌드 파일**: Windows용 실행 파일 (.exe) 또는 Android용 APK
